The graph indicates temporal variation of light intensity in an office in Halifax, Nova Scotia. The measurements were done by DFRobot Ambient Light Sensor connected to an Arduino Uno microcontroller, which is subject to a controlling program that supplies real-time data to a computer over a USB interface. The computer uses a C program to capture the serial stream, calculate statistical measures in 1-minute intervals, and append the results to a file on the computer. At 10 minute intervals, an R program reads the file and creates the image shown above.

Once per day, sunrise and sunset are inferred from the light intensity variation, and compared with actual times from celestial calculations; see the diagram and R code.

This project is developed in the open on github.com.
