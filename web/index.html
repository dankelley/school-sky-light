The black line is the mean in each minute. The data are light measurements made a DFRobot Ambient Light Sensor connected to an Arduino Uno microcontroller, which is driven by controlling program that places the data onto a serial port running through a USB interface. A C program captures this serial stream, time-stamps the data, and forms statistical measures at 1-minute intervals. At 10 minute intervals, an R program reads these data and creates the image shown above.
